# 🎯 FirstIssue

> Finding a "good first issue" shouldn't be harder than fixing it. FirstIssue is a curated, Tinder-style matchmaker for open-source contributions.

## The Core Concept
Getting into open source is intimidating. Developers are told to "find a good first issue," but manually filtering through GitHub is overwhelming. FirstIssue aggregates beginner-friendly issues from across GitHub and presents them in a clean, interactive UI. 

## MVP Features Planned
- GitHub OAuth Authentication
- Curated Issue Feed (fetching from GitHub REST API `/search/issues`)
- Language Filtering Engine
- "Save for Later" Bookmarking System
